# Agent: codi-api-designer

> API design agent for ensuring MCP server endpoints follow best practices.

## When to Delegate

- MCP server needs API design review for naming, pagination, and error handling
- The user wants to validate the MCP server's tool interface design
- Complex API surfaces need structured design review

## How to Use

Defined at `.codi/agents/codi-api-designer.md`. Invoke via the Agent tool with `subagent_type` set to `codi-api-designer`.

## Key Capabilities

- REST and GraphQL API design review
- Naming conventions, pagination, and error handling best practices
- Schema-first design validation
