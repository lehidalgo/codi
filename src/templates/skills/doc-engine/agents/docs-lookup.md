# Agent: codi-docs-lookup

> Research agent for verifying content accuracy during document generation.

## When to Delegate

- Document generation needs fact-checking or API reference verification
- The user wants to include accurate technical details in reports
- Research is needed to support document content with real data

## How to Use

Defined at `.codi/agents/codi-docs-lookup.md`. Invoke via the Agent tool with `subagent_type` set to `codi-docs-lookup`.

## Key Capabilities

- Multi-source documentation search
- API signature and deprecation verification
- Working example discovery from official sources
