# Agent: codi-test-generator

> Test generation agent for web application testing scenarios.

## When to Delegate

- Webapp testing identifies areas needing automated test coverage
- The user wants to generate Playwright or browser-based tests
- Test suites need expansion based on webapp testing results

## How to Use

Defined at `.codi/agents/codi-test-generator.md`. Invoke via the Agent tool with `subagent_type` set to `codi-test-generator`.

## Key Capabilities

- TDD RED-GREEN-REFACTOR workflow
- Framework auto-detection (Jest, Vitest, Playwright)
- Browser-based and integration test creation
