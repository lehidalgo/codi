## Usage Guidelines

- Use values imagery in internal presentations, onboarding materials, and corporate communications
- The overview images show all three values together — use for cover slides or summary sections
- Individual value images are for deep-dive slides or section headers
- Spanish versions are the default for BBVA Spain communications
- English versions are for international audiences
- Both plain and microillustration variants are available — choose based on formality level
