# Agent: codi-docs-lookup

> Research agent that searches documentation before coding to verify API signatures and find examples.

## When to Delegate

- Documentation needs API signature verification or deprecation checks
- The user wants to find working examples from official docs
- Research is needed before writing or updating documentation

## How to Use

Defined at `.codi/agents/codi-docs-lookup.md`. Invoke via the Agent tool with `subagent_type` set to `codi-docs-lookup`.

## Key Capabilities

- Multi-source documentation search
- API signature and deprecation verification
- Working example discovery from official sources
